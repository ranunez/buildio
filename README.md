## Buildio
![Image](Shared/Assets.xcassets/AppIcon.appiconset/100.png) 

Unofficial [Bitrise CI](https://bitrise.io) client for iOS and macOS

Lightweight, multiaccount support, notifications and more

Written on Swift + SwiftUI
